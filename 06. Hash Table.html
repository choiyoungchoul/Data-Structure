Hash Table (해쉬 테이블) 2. 특징 및 용어 특정 Key에 데이터를(Value)를 저장하는 구조 파이썬의 경우 딕셔너리(Dictionary) 타입이 있다. 해쉬 (Hash) : 임의 값을 고정 길이로 변환 하는 것. 해쉬 테이블 (Hash Table) : 키 값으로 접근이 가능한 데이터 구조 슬롯 (Slot) : 한 개의 데이터를 저장할 수 있는 공간 3. 소스(파이썬) hash_table = list([i for i in range(10)]) * 리스트 컴프레이션 -> 반복문을 LIST 안에 써서 0 ~ 9까지 데이터를 삽입 hash_table * 해쉬 함수 디비전(Division)법 -> 나누기를 통한 나머지 값을 사용하는 기법 def hash_func(key): return key % 5 data1 = 'Andy' data2 = 'Dave' data3 = 'Trump' print (ord(data1[0])) * ord 함수는 아스키 값을 표시(리턴) -> 제일 앞글자 아스키 코드를 Key 값으로 활용 하기위해 def storage_data(data, value): * 저장함수 key = ord(data[0]) hash_address = hash_func(key) hash_table[hash_address] = value *함수에 값 저장 storage_data('Andy','01055553333') storage_data('Dave','01044443333') storage_data('Trump','01022223333') *출력 함수 def get_data(data): key = ord(data[0]) hash_address = hash_func(key) return hash_table[hash_address] *함수를 이용해 출력 get_data('Andy')
