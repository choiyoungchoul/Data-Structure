1. 스택(Stack) 자료를 한쪽으로만 IN, OUT 할 수 있는 구조 가장 나중에 들어오는 데이터가 가장 먼저 나가는 LIFO(Last in, First out) 구조 보통 재귀함수 형식으로 사용 2. 특징 및 주요 함수 장점 : 구조가 단순, 데이터 저장,읽기 속도가 빠름 단점 : 저장공간 낭비 가능성 높음 push(): 데이터를 IN pop() : 데이터를 OUT 3. 소스 (제공해주는 함수를 사용하지 않고 구현 (파이썬)) stack_list = list() def push(data): stack_list.append(data) def pop(): data = stack_list[-1] del stack_list[-1] return data ------------------------------------------------- for index in range(10): push(index) pop() 9 8 7 . . . * 파이썬의 리스트는 인덱싱이 0부터 시작하므로, -1 인덱스는 리스트의 마지막 요소
